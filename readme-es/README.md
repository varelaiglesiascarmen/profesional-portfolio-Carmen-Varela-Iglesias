## Presentación
Este repositorio contiene mi Portfolio Profesional, un sitio web personal que he diseñado y desarrollado completamente utilizando **Bootstrap 5, HTML, CSS y JavaScript**. Su objetivo es mostrar de forma clara y profesional mis proyectos, habilidades y trayectoria como desarrolladora Frontend.

Siempre me ha apasionado el arte, y he descubierto que cuando se combina con la tecnología se abre un mundo sin límites. Me caracterizo por construir **interfaces limpias, eficientes y visualmente atractivas**, combinando buenas prácticas de desarrollo con atención al diseño y la experiencia de usuario.
Este portfolio lo desarrollo para presentarlo en procesos de selección, mostrando mi trabajo, mis conocimientos y mi estilo de desarrollo.

## Objetivo del Portfolio

- Mostrar proyectos propios, académicos o personales relevantes.
- Evidenciar habilidades en HTML, CSS, JavaScript y diseño responsivo con Bootstrap.
- Reflejar mi identidad profesional: cuidado por el detalle, código organizado y experiencia de usuario.
- Servir como carta de presentación ante reclutadores o posibles colaboradores.

## Tecnologías usadas

- Bootstrap 5 para diseño responsivo, componentes reutilizables y UI moderna.
- HTML5 y CSS3 para estructura semántica, estilos y diseño adaptativo.
- JavaScript (vanilla) para añadir interactividad cuando es necesario.
- Git y GitHub para control de versiones, organización de código y gestión del repositorio.

## Cómo usar o ver el portfolio localmente

Actualmente el sitio solo puede visualizarse de manera local, ya que aún me encuentro desarrollando y ajustando el código. Una vez finalizado, el proyecto será desplegado mediante GitHub Pages.

Para visualizarlo localmente:

1. Clona este repositorio:
    git clone https://github.com/varelaiglesiascarmen/profesional-portfolio-Carmen-Varela-Iglesias.git
   
2. Entra en la carpeta:
    cd profesional-portfolio-Carmen-Varela-Iglesias

3. Abre el archivo index.html en tu navegador.

El sitio se renderizará localmente sin necesidad de configuración adicional.
